# Examples of C and sqlite cooperation

### Рассмотрены следующие примеры: ###
- подключение БД;
- создание таблицы;
- INSERT operation;
- SELECT operation;
- UPDATE operation;
- DELETE operation;
- параметризованные запросы;
- вставка изображения в БД;
- вставка данных в режиме autocommit и в виде транзакций;
- вывод метаданных базы.
